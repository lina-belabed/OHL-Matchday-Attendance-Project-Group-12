# OHL Matchday Attendance Prediction — Group 12

Predicting home match attendance at OH Leuven (OHL) using historical match data, weather, media buzz, and team form. Built for operational planning — staffing, catering, and promotional decisions.

---

## What this project does

Given a set of match conditions (opponent, timing, team form, weather, promotions), the models forecast how many fans will show up at Den Dreef. Three estimates are produced:

- **Expected attendance** — best single-number estimate (Ridge regression)
- **Conservative estimate** — 65% of past matches exceeded this (QR35)
- **Minimum planning figure** — 75% of past matches exceeded this (QR25)

## Web app

The app has two modes switchable from the sidebar:

- **Simple mode** (default) — designed for non-technical users like the marketing team. Replaces raw statistical inputs with plain-English options (e.g. "Expected online buzz: Low / Normal / High / Very High"). Opponent strength and weekend/week detection are filled in automatically.
- **Advanced mode** — exposes all raw model inputs for the data team.

### Key inputs

| Input | Notes |
|-------|-------|
| Opponent | Opponent historical strength is auto-filled |
| Match date | Weekend and calendar week derived automatically |
| Kickoff time | |
| Matchday | 1–33 |
| OHL form | Points, win rate, season points |
| Head-to-head | Last result and goal difference vs this opponent |
| Recent attendance | 3-game average — strongest predictor |
| Promotions | Ticket promotion and/or marketing campaign active |
| Media coverage | Press articles + online buzz level |
| Weather | Conditions and temperature |

## Models

| Model | Test MAE | Notes |
|-------|---------|-------|
| Ridge Regression | ~1,022 tickets | Best generalizer — used for point estimate |
| ElasticNet | ~1,069 tickets | |
| Quantile (q=0.35) | ~956 tickets | Conservative lower bound |
| Quantile (q=0.25) | ~916 tickets | Very conservative lower bound |
| Random Forest | ~1,592 tickets | Overfits on small dataset |
| Gradient Boosting | ~1,478 tickets | Overfits on small dataset |

**Train/test split:** time-based — trained on 2022/23–2024/25 seasons (57 matches), evaluated on 2025/26 (14 matches).

### Why is the error ~1,000 tickets?

The dataset has 71 home matches total. With this few samples, the models generalize conservatively. The 2025/26 season also showed an 18% drop in average attendance compared to training years — a structural shift the models weren't trained on.

## Top predictors (in order of importance)

1. **Recent home attendance** — 3-match rolling average
2. **Season phase** — late-season matches draw significantly more fans
3. **OHL form** — points per game this season
4. **Opponent** — big clubs (Club Brugge, Anderlecht) draw 9–10k; lower-table teams ~4–6k
5. **Ticket promotions** — active promos increase attendance
6. **Weekend vs weekday** — weekday matches draw fewer fans
7. **Online buzz** — Google Trends interest in the week before the match

## Data sources

Six raw files merged during cleaning:
- Match results and schedule (`gold_match.csv`)
- Ticket sales breakdown (`gold_match_tickets.csv`)
- Contextual flags — holidays, promotions, campaigns
- Weather data — temperature, rain, wind, sunshine
- Press article counts (7 days pre-match)
- Google Trends interest scores (7-day averages)

---

## Limitations

- **Small dataset** — 71 home matches across 4 seasons
- **~±1,000 ticket error** on held-out test data
- **Distribution shift** — 2025/26 attendance is structurally lower than training seasons
- **Manual inputs** — weather, Google Trends, and recent attendance must be filled in by hand (no live API integration)
- Use predictions as **directional guidance**, not precise forecasts
